# Homebrew Tap for Fluxzy

This is the official Homebrew tap for [Fluxzy](https://www.fluxzy.io).

## Installation

```bash
brew tap haga-rak/fluxzy
brew install fluxzy
```

## Usage

After installation, you can use the `fluxzy` command:

```bash
fluxzy --help
```
